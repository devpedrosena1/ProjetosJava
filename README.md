# Projeto em Java 🎮

*Este repositório contém uma coleção de projetos desenvolvidos em Java durante aulas e práticas de programação. Cada pasta representa um projeto ou exercício específico, identificado pela data ou pelo tema abordado.*

<br>

**Estrutura do Repositório**
- AulaJava25_03: Projeto desenvolvido na aula de 25 de março.​

- AulaJava_21_03: Projeto desenvolvido na aula de 21 de março.​

- ProjetoAluno: Projeto relacionado à gestão de alunos.​

- ProjetoTesteCP/ProjetoCP: Projeto de teste denominado "ProjetoCP".​

- aula01_04/ProjetoCRUD: Projeto CRUD desenvolvido na aula de 1º de abril.​

- aula103/ProjetoMatriz: Projeto relacionado a operações com matrizes.​

- aula10_02/Main: Projeto principal da aula de 10 de fevereiro.​

- aula12-02/ProjetoProduto: Projeto focado na gestão de produtos, desenvolvido em 12 de fevereiro.​

- aula17_02/ProjetoProduto: Outra versão do projeto de produtos, desenvolvida em 17 de fevereiro.​

- aula18_03/ProjetoDAO: Projeto que implementa o padrão DAO (Data Access Object), desenvolvido em 18 de março.​

- aula19-02: Projeto desenvolvido na aula de 19 de fevereiro.​

- aula24-02/aula/aula24-02: Projeto relacionado à aula de 24 de fevereiro.​

- aula26-02/ProjetoAlunoConexao: Projeto que envolve a conexão de dados de alunos, desenvolvido em 26 de fevereiro.​

- aulaJava07_03: Projeto desenvolvido na aula de 7 de março.

<br>

**Pré-requisitos**
Para executar os projetos deste repositório, é necessário ter instalado:

-- Java Development Kit (JDK) (versão 11 ou superior).

-- Inteliij IDEA ou outra IDE compatível com Java.
